Clinical Overview

What is Brain Tumor Detection?

Combining magnetic resonance scans and deep convolution layers to accelerate neurologists' analysis workflows.

Why are MRI scans used?

Brain tumors represent severe, life-altering pathologies that manifest in soft cranial structures. To analyze these anomalies without surgical invasiveness, clinicians utilize Magnetic Resonance Imaging (MRI).

MRI scanning provides soft-tissue contrast that highlights spatial changes, lesions, and abnormal growths. Using deep learning networks, the detection process can be augmented, identifying pixel-level anomalies that might escape manual initial review.

How does Deep Learning help?

Deep convolutional neural networks automatically extract hierarchy levels of visual information (from low-level edges to high-level tissue contours). This eliminates manual feature engineering, providing rapid and consistent scan assessments on demand.

Model Details & VGG16

This system utilizes a pre-trained **VGG16 model** that was originally trained on millions of generic images. By freezing the early spatial feature extraction layers and fine-tuning custom classification layers specifically with brain MRI images, the network is optimized to identify subtle pathological features in MRI scans.

TensorFlow & Keras Implementation The Keras model is saved as a .h5 file. It pre-processes uploads dynamically by resizing images to 128x128 pixels, converting them to 3-channel RGB, and scaling pixel elements to a [0, 1] float range.

Class 1
Meningioma

Tumors that develop in the meninges, the protective membranes surrounding the brain and spinal cord.

Class 2
No Tumor

Diagnostic scans verifying healthy, normal brain anatomy without visual pathological masses.

Class 3
Pituitary

Tumors that grow in the pituitary gland at the base of the brain, affecting endocrine systems.
